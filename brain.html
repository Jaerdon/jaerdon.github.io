Background

In late 2018, I had an MRI scan done. Recently, I requested the data from the MRI on a disc. The data is stored in a format called DICOM, which has become the standard for medical imaging. There are a number of tools created specifically for the purpose of viewing DICOM data, I chose to use 3D Slicer, which is a fantastic open-source tool that can turn DICOM series into 3D models. It was printed on my friend Tim's (heavily modded) Maker Select v2. The model isn't perfect, but for the amount of time I had to create it I am happy with the results.
