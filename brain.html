Background

I recently acquired the data from an MRI scan I had done in 2018, stored in a format called DICOM. As DICOM has become the standard for medical imaging, there are a number of tools created specifically for the purpose of viewing DICOM data. This model was created using 3D Slicer, printed on my friend Timothy's (heavily modded) Maker Select v2.
