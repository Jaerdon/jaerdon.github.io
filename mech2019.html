Mechatronics 2019 Demo




Stack Magazine and Conveyor Distribution Demo

This project is was created as part of a project my friend Jennah Bari and I worked on to demonstrate Mechatronics principles. Mechatronics is a branch of engineering which focuses on unifying mechanical engineering, electrical engineering, control systems, and computer science into one cohesive system.

This demonstration shows an example industrial system with cylindrical parts placed onto the conveyor via a stack magazine. The system detects the presence of a cylinder using an optical sensor, which is triggered when an object breaks a light barrier. Once the conveyor has been activated, the part passes underneath a color sensor, which detects the color of the block and determines what operations to perform on the part. If the part is silver, the system performs a punch operation on the part, then sorts it by lowering a diverter. If the part is purple or red, the system simply sorts the part by activating the same diverter solenoid. If the part is blue or black, the part is considered defective and sorted into the discard bin for recycling. The cycle repeats for a preset number of cycles.
